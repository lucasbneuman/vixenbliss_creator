# Context Index

Este directorio centraliza contexto operativo para Codex sin duplicar `/AGENTS.md`.

## Archivos

- `docs/context/governance.md`: gobernanza detallada (modos, checklists, enforcement).
- `docs/external-systems/README.md`: indice de dependencias externas.

## Regla de uso

1. Leer primero `/AGENTS.md` (router global).
2. Abrir despues `docs/context/governance.md` para detalle operativo.
3. Validar contratos en `docs/api-contracts/*`.
4. Si hay proveedor externo involucrado, abrir `docs/external-systems/*`.
