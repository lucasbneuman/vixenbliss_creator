# Archive

## Audiencia

- developers
- agentes solo para consulta historica puntual

## Vigencia

- `archivado`

## Que vive aca

- documentos reemplazados por versiones mas alineadas al codigo actual
- analisis historicos de arquitectura y proveedores
- material que conserva contexto, pero ya no gobierna decisiones actuales

## Regla de uso

Nada en `docs/99-archive/` debe tratarse como fuente de verdad activa.

Si un documento archivado tiene reemplazo, debe indicarlo al inicio.
